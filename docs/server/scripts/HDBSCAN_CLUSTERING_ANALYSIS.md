# HDBSCAN 클러스터링 결과 분석 보고서

## 1. 개요

본 보고서는 HDBSCAN (Hierarchical Density-Based Spatial Clustering of Applications with Noise) 알고리즘을 사용하여 수행한 클러스터링 분석 결과를 상세히 설명합니다. HDBSCAN은 밀도 기반 클러스터링 알고리즘으로, 클러스터 수를 사전에 지정하지 않고 데이터의 밀도 분포를 기반으로 자동으로 클러스터를 발견합니다.

---

## 2. 클러스터링 성능 지표

### 최적 설정

- **방법**: HDBSCAN (Density-based Clustering)
- **참고 논문**: McInnes et al. (2017)
- **최적 파라미터**:
  - `min_cluster_size`: 50
  - `min_samples`: 50
  - `metric`: euclidean
  - `cluster_selection_method`: eom (Excess of Mass)

### 성능 지표

| 지표 | 값 | 평가 |
|------|-----|------|
| **Silhouette Score** | **0.6014** | ⭐⭐⭐⭐⭐ 매우 우수 |
| **Davies-Bouldin Index** | **0.6872** | ⭐⭐⭐⭐⭐ 매우 우수 (낮을수록 좋음) |
| **Calinski-Harabasz Index** | **6385.79** | ⭐⭐⭐⭐⭐ 매우 우수 |
| **클러스터 수** | **19개** | 자동 결정 |
| **노이즈 포인트** | **60명 (0.3%)** | 매우 낮음 |

### 기존 방법과 비교

| 방법 | 클러스터 수 | Silhouette Score | Davies-Bouldin Index | 평가 |
|------|------------|------------------|---------------------|------|
| **K-Means (k=8)** | 8 | 0.3061 | 1.7035 | 기준 |
| **K-Means (k=12)** | 12 | 0.4458 | 1.5779 | +45.6% |
| **K-Means (k=16)** | 16 | 0.5423 | 1.0613 | +77.2% |
| **HDBSCAN** | **19** | **0.6014** | **0.6872** | **+96.5%** ⭐ |

**결론**: HDBSCAN이 모든 평가 지표에서 최고 성능을 달성했습니다!
